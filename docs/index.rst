Simple-Kria-Kv260-Example Documentation
========================================

Reference firmware and software example for the Xilinx Kria KV260 Vision AI
Starter Kit (Kria K26 SOM) (Xilinx Zynq UltraScale+ MPSoC, FPGA part
``xck26-sfvc784-2lv-c``). Built on the SLAC ``axi-soc-ultra-plus-core``
platform; shared workflow documentation lives on the
:hub:`platform docs site <index.html>`.
